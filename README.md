# MASLD liver-biopsy transcriptomic axes

This repository provides a reproducible analysis pipeline for deriving compact transcriptomic axes
from public liver-biopsy data and evaluating their association with advanced fibrosis across
independent GEO cohorts.

## Repository scope

This repository is limited to computational reproducibility:

- Source scripts needed to download public data, process expression matrices, run analyses, and regenerate tables/figures.
- Public data provenance and decision rules needed to understand the generated outputs.
- Reference output tables and figures that can be regenerated by the pipeline.
- A supplementary workbook containing the complete table set used to support the article.

It does not include article text, editorial files, private planning files, or local development scaffolding.

## Quick start (one command)

```bash
bash scripts/reproduce_one_click.sh
```

## What the pipeline does

- Downloads required public inputs on demand (GEO series matrices; selected GEO supplementary files; NCBI gene info; GPL annotations).
- Extracts fibrosis staging metadata and defines advanced fibrosis as stage ≥F3 versus ≤F2 where unambiguous staging is available.
- Treats labels spanning the F3 threshold as missing for the primary endpoint rather than forcing them into either endpoint class.
- Uses GSE163211 as the targeted-panel discovery cohort and excludes it from the primary F3+ validation endpoint.
- Derives compact NMF-based transcriptional axes and transfers fixed signatures across validation cohorts.
- Records platform-specific expression transformations before within-cohort signature scoring.
- Estimates cohort-wise odds ratios per 1-SD score and pools them with REML/Hartung-Knapp random-effects synthesis.
- Benchmarks the transferred axis against published MASLD/MASH/fibrosis transcriptomic signatures.
- Evaluates cohort-level transportability with leave-one-cohort-out discrimination, calibration, and probability accuracy.
- Annotates signatures with pathway enrichment and an external human liver single-cell RNA-seq atlas.

## Outputs

- Derived tables: `results/`
- Publication-ready figures: `plots/publication/`
- Complete supplementary workbook: `supplementary_tables/SUPPLEMENTARY_TABLES.xlsx`
- Provenance map: `docs/FIGURE_PROVENANCE.tsv`
- Supplementary-table provenance map: `docs/TABLE_PROVENANCE.tsv`
- Expression-processing summary: `docs/EXPRESSION_PROCESSING.tsv`

## System requirements

- Python 3.10+
- macOS/Linux recommended

## Notes

- First run may take time due to public downloads.
- No raw private data is included; all inputs are downloaded from public sources listed in `docs/DATA_MANIFEST.tsv`.
- The curated He et al. signature-definition table at `data/references/prior_signatures/he_2023_signature_definitions.tsv` provides the published gene sets used for benchmarking.
